# spiderfucker-python

[SpiderFucker](https://github.com/FastGitORG/SpiderFucker/) 的 Python 实现。

## 用途

根据规则允许和阻止爬虫。

## 使用

```bash
python3 classify.py
```

然后查看 `result` 文件夹。
